26 ## Usage
27 This microservice is created automatically on changes to this repository via
28 Docker's public registry for:
29 * [x86_64](https://hub.docker.com/r/chalmersrevere/opendlv-signal-viewer-amd64/tags/)
30 * [armhf](https://hub.docker.com/r/chalmersrevere/opendlv-signal-viewer-armhf/tags/)
31 
32 To use this microservice for viewing any messages from the OpenDLV Standard
33 Message Set that are exchanged in a running OpenDLV.io session (running at
34 111 in the example), simply run it as follows:
35 
36 ```
37 docker run --rm --net=host -p 8080:8080 chalmersrevere/opendlv-signal-viewer-multi:v0.0.4 --cid=111
38 ```
39 
40 Now, simply point your web-browser to the IP address and port 8080 where you
41 started this microservice to see any currently exchanged messages:
42 
43 ![screenshot from signal viewer](https://raw.githubusercontent.com/chalmers-revere/opendlv-signal-viewer/master/signal-viewer.gif)
